# Code of Conduct

This project expects respectful, constructive participation.

## Expected behavior

- Be considerate and professional.
- Focus feedback on ideas and work, not people.
- Assume good intent, and ask for clarification when needed.

## Unacceptable behavior

Harassment, threats, discrimination, and abusive comments are not acceptable.

## Reporting

Report unacceptable behavior to `<MAINTAINER_EMAIL>`. Maintainers will review reports and take appropriate action.
